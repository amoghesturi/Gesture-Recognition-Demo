Youtube Gesture Control Demo

This is a simple demo web page that uses hand gestures to control the youtube video below.

Instructions:

  • Grant permission for the web page to use the web camera. Try using newer versions of chrome or firefox
  • To play/pause the video, make a hand gesture of closed fist in front of the camera
  • To reset any settings, just refresh the page
